Stand-alone программное обеспечение для совместного редактирования геоданных

NextGIS Crowd позволяет совместно редактировать расположение и описание точек, наборов атрибутов может быть любым. NextGIS Crowd использовался в краудсорсинговом проекте УИК ГЕО для сбора данных по десяткам тысяч местоположений участковых избирательных комиссий.

NextGIS Crowd

NGC - это многопользовательский редактор геоданных (пока только точки).

После развертывания на сервере, пользователь может редактировать местоположение точек и структурированную описательную информацию. Для редактирования можно привлекать других пользователей, они должны пройти простую систему регистрации встроенную в проект. В проекте "из коробки" доступны удобные механизмы поиска и выборок объектов.

Исходный код

Код NextGIS Crowd написан на Python (серверная) и JavaScript (клиентская часть) и опубликован под открытой лиценцией GNU GPL version 2 (подробнее). Системные требования: Python 2.7+, spatialite, GEOS, PROJ.4, Virtualenv.

Код предоставляется “as is”, присылайте ваши pull-реквесты.

О проекте

NextGIS Crowd (NGC) - это простое средство для совместной работы с геоданными (точками на карте). Установите его у себя, загрузите туда данные для редактирования и попросите у других помощи. Помощникам нужно будет только зарегистрироваться, NGC работает в браузере. Вместе вы сможете успеть гораздо больше.
